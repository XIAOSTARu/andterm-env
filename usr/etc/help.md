# ====使用说明====

1. 如何给 MT管理器 终端添加环境？
在终端中执行以下命令↓↓↓
`/data/adb/andterm_env/usr/bin/install-andtermenv "/data/user/0/bin.mt.plus/files/term/etc/bash.bashrc"`

2. 如何在mt管理器移除环境？
在终端中执行以下命令↓↓↓
`uninstall-andtermenv "/data/user/0/bin.mt.plus/files/term/etc/bash.bashrc"`

3. 如何安装软件？
在终端中执行以下命令↓↓↓
`install (链接,deb格式, 必须是termux的包)`

4. 如何卸载软件？
暂时还没有专门卸载的，只能自行删除
